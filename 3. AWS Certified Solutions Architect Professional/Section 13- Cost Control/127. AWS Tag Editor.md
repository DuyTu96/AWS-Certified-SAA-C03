# 127. AWS Tag Editor

## 🎯 Giới thiệu
AWS Tag Editor là công cụ giúp bạn:
- Tìm các AWS resources để gắn tag
- Quản lý tag cho **nhiều resources cùng lúc**
- Thực hiện các thao tác như **add**, **update**, **delete** tags
- Tìm kiếm **tagged** hoặc **untagged resources** trên **tất cả AWS regions**

## 1. Chức năng chính của Tag Editor
- Cho phép quản lý tags trên nhiều resources cùng lúc
- Hữu ích khi cần chỉnh sửa tag hàng loạt thay vì từng resource riêng lẻ
- Có thể **add**, **update**, và **delete** tags

## 2. Khả năng tìm kiếm resource
- Có thể tìm các resources cần gắn tag
- Có thể tìm:
  - **tagged resources**
  - **untagged resources**
- Phạm vi tìm kiếm áp dụng trên **all AWS regions**

## 3. Ý nghĩa trong kỳ thi AWS
- Đây là một chủ đề ngắn nhưng vẫn có thể xuất hiện trong **AWS exam**
- Cần nhớ trọng tâm là:
  - quản lý tag
  - nhiều resource cùng lúc
  - tìm tagged/untagged resources
  - trên toàn bộ regions
