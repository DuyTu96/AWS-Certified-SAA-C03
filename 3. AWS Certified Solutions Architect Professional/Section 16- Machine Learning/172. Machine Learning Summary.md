# 172. Machine Learning Summary

## 🎯 Giới thiệu
AWS có một nhóm **machine learning services** quan trọng mà bạn cần nhớ để đi thi. Nội dung của bài này chủ yếu là nhận diện đúng service nào làm nhiệm vụ gì trong hệ sinh thái AWS.

## 1. Các dịch vụ machine learning trên AWS
- **Rekognition**: dùng cho **face detection**, **labeling**, và **celebrity recognition**.
- **Transcribe**: dùng để chuyển **audio into text**, tạo **subtitles**.
- **Polly**: chức năng ngược lại với Transcribe, dùng để chuyển **text into audio**.
- **Translate**: dùng cho **translations**.
- **Lex**: dùng để xây dựng **conversational bots** hoặc **chatbots**.
- Khi kết hợp **Lex** với **Connect**, có thể tạo **cloud contact center**.
- **Comprehend**: dùng cho **natural language processing**.
- **SageMaker**: là một **fully featured machine learning service**, dành cho **developer** và **data scientist**.
- **Kendra**: là một **ML-powered document search engine**.
- **Personalize**: dùng cho **real-time personalized recommendations** cho customer.
- **Textract**: dùng để **detect text** và **extract data** từ nhiều loại tài liệu.

## 2. Cách ghi nhớ nhanh theo chức năng
- **Nhìn ảnh / nhận diện**: `Rekognition`
- **Âm thanh -> văn bản**: `Transcribe`
- **Văn bản -> âm thanh**: `Polly`
- **Dịch ngôn ngữ**: `Translate`
- **Chatbot / bot hội thoại**: `Lex`
- **Xử lý ngôn ngữ tự nhiên**: `Comprehend`
- **Nền tảng ML đầy đủ**: `SageMaker`
- **Tìm kiếm tài liệu bằng ML**: `Kendra`
- **Gợi ý cá nhân hóa theo thời gian thực**: `Personalize`
- **Trích xuất text/data từ tài liệu**: `Textract`

## 3. Điểm cần nhớ cho kỳ thi
- AWS thường kiểm tra khả năng **match đúng service với use case**.
- Các cặp dễ nhầm:
  - **Transcribe** vs **Polly**
  - **Rekognition** vs **Textract**
  - **Lex** vs **Connect**
- Học theo chức năng thực tế sẽ dễ nhớ hơn học theo tên service đơn lẻ.

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| Rekognition | Face detection, labeling, celebrity recognition |
| Transcribe | Audio to text, subtitles |
| Polly | Text to audio |
| Translate | Translation |
| Lex | Conversational bots, chatbots |
| Connect + Lex | Cloud contact center |
| Comprehend | Natural language processing |
| SageMaker | Fully featured machine learning service |
| Kendra | ML-powered document search engine |
| Personalize | Real-time personalized recommendations |
| Textract | Detect text và extract data từ documents |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Học theo cặp đối nghịch:
  - **Transcribe** = nghe thành chữ
  - **Polly** = chữ thành tiếng
- Nhớ nhóm dịch vụ theo mục tiêu:
  - **Rekognition / Textract** = xử lý nội dung từ hình ảnh, tài liệu
  - **Lex / Connect** = giao tiếp với khách hàng
  - **Comprehend / Translate** = ngôn ngữ
  - **Kendra / Personalize** = tìm kiếm và cá nhân hóa
- Nếu đề bài nhắc đến **ML-powered document search**, nghĩ ngay đến **Kendra**.
- Nếu đề bài nhắc đến **real-time recommendations**, nghĩ ngay đến **Personalize**.

## ✅ Kết luận
Bài học này tập trung vào việc nhận diện nhanh các **AWS machine learning services** và chức năng chính của chúng. Với kỳ thi AWS, mục tiêu quan trọng nhất là ghép đúng service với đúng tình huống sử dụng.
