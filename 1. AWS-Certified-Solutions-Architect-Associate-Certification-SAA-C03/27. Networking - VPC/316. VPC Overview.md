# 316. VPC Overview

## 🎯 Giới thiệu
- **VPC** là viết tắt của **Virtual Private Cloud**.
- Trong **một AWS region**, bạn có thể tạo **nhiều VPC**.
- Giới hạn mặc định là **5 VPC per region**, nhưng đây là **soft limit** nên có thể tăng.
- Mỗi VPC có thể được gán tối đa **5 CIDRs**.

## 1. CIDR trong VPC
- Mỗi **CIDR** trong VPC có:
  - **Kích thước tối thiểu**: `/28` tương ứng ít nhất **16 IP addresses**
  - **Kích thước tối đa**: `/16` tương ứng tối đa **65,536 IP addresses**
- Vì VPC là tài nguyên **private**, chỉ dùng được **Private IPv4 ranges**.

## 2. Chọn dải IP cho VPC
- Bạn có thể chọn dải IP theo nhu cầu, nhưng cần đảm bảo:
  - **Không overlap** với các VPC khác
  - **Không overlap** với các network khác
  - **Không overlap** với corporate network
- Lý do là khi cần **connect** các network với nhau, IP ranges phải **không trùng nhau** để tránh xung đột địa chỉ.

## 3. Ý chính cần nhớ cho hands-on
- Mục tiêu của phần thực hành là tạo một **simple VPC** trong một region.
- Trọng tâm của bài này là hiểu:
  - VPC là gì
  - Giới hạn về số lượng VPC và CIDR
  - Quy tắc chọn CIDR không trùng lặp

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| VPC | Virtual Private Cloud |
| VPC per region | Tối đa 5, là soft limit |
| CIDR per VPC | Tối đa 5 |
| CIDR nhỏ nhất | `/28` = 16 IP addresses |
| CIDR lớn nhất | `/16` = 65,536 IP addresses |
| Loại IP | Chỉ Private IPv4 ranges |
| Quy tắc chọn CIDR | Không được overlap với VPC/network/corporate network khác |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Nhớ 3 điểm hay hỏi trong đề thi: **VPC = private**, **5 VPC per region (soft limit)**, **5 CIDRs per VPC**.
- CIDR trong VPC phải nằm trong **Private IPv4 ranges**.
- Khi kết nối nhiều network, ưu tiên kiểm tra **CIDR overlap** trước.
- Nhớ mốc kích thước CIDR:
  - `/28` = nhỏ nhất
  - `/16` = lớn nhất

## ✅ Kết luận
- VPC là nền tảng mạng riêng trong AWS và có thể tồn tại nhiều VPC trong cùng một region.
- Khi thiết kế VPC, cần chú ý giới hạn số lượng, số CIDR, kích thước CIDR và đặc biệt là **không để IP ranges bị overlap**.
