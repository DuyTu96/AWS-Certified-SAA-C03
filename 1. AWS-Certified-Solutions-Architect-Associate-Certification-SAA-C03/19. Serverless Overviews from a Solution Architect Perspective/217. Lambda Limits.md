# 217. Lambda Limits

## 🎯 Giới thiệu
Lambda có một số **limits** rất quan trọng cần nhớ trước kỳ thi AWS, vì đề thi thường kiểm tra các giới hạn này. Các **limits** trong transcript đều là **per region**.

## 1. ⚙️ Execution Limits
- **Memory allocation**: từ **128 MB** đến **10 GB**
- Mức tăng bộ nhớ: **64 MB increments**
- Khi tăng memory, Lambda sẽ có thêm **vCPU**
- **Maximum execution time**: **900 seconds** = **15 minutes**
- **Environment variables**: tối đa **4 KB**
- **Temporary storage** trong **/tmp**: tối đa **10 GB**
- **Concurrent executions**: tối đa **1000**
  - Có thể tăng thêm nếu gửi **request**
  - Nên dùng **reserved concurrency** sớm nếu cần kiểm soát

## 2. 📦 Deployment Limits
- **Compressed zip** tối đa: **50 MB**
- **Uncompressed** tối đa: **250 MB**
- Nếu file quá lớn, nên dùng **/tmp** thay vì cố nhét vào deployment package

## 3. 🧠 Ý nghĩa khi làm bài thi
- Nếu workload cần:
  - **30 GB RAM**
  - **30 minutes execution time**
  - **big file 3 GB**
- Thì đó **không phải** use case phù hợp cho **Lambda**

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| Phạm vi | Các limits là **per region** |
| Memory | **128 MB - 10 GB**, tăng theo **64 MB** |
| vCPU | Tăng theo memory |
| Execution time | Tối đa **900 seconds / 15 minutes** |
| Environment variables | Tối đa **4 KB** |
| /tmp storage | Tối đa **10 GB** |
| Concurrent executions | Tối đa **1000** |
| Deployment zip | **50 MB compressed** |
| Deployment size | **250 MB uncompressed** |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Nhớ cặp số quan trọng: **128 MB - 10 GB**, **15 minutes**, **4 KB**, **10 GB /tmp**, **1000 concurrent**, **50 MB zip**, **250 MB uncompressed**
- Nếu đề bài đòi **file lớn**, **RAM lớn**, hoặc **thời gian chạy dài**, hãy nghĩ ngay: **Lambda không phù hợp**
- **reserved concurrency** là chi tiết dễ bị hỏi khi đề cập giới hạn thực thi

## ✅ Kết luận
**Lambda Limits** là phần rất dễ ra thi. Chỉ cần nhớ đúng các mốc về **memory**, **execution time**, **environment variables**, **/tmp**, **concurrency**, và **deployment size** là có thể loại nhanh các câu hỏi chọn dịch vụ phù hợp.
