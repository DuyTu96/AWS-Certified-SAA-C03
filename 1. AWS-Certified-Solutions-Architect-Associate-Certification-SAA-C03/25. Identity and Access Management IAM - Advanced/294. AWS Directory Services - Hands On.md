# 294. AWS Directory Services - Hands On

## 🎯 Giới thiệu
- Bài học này giới thiệu nhanh các lựa chọn trong **AWS Directory Services** khi vào console và gõ `directory service`.
- Có **4 options**, nhưng **Amazon Cognito User Pool** thực chất chuyển sang dịch vụ **Cognito**, nên **không tính** vào Directory Services.
- Trọng tâm là hiểu vai trò của từng dịch vụ để ôn thi AWS.

## 1. AWS Managed Microsoft AD
- Đây là **Active Directory** được tích hợp với **AWS Cloud**.
- Có thể tạo **trust relationship** với **on-premise directory**.
- Có 2 edition:
  - **Standard Edition**: tối đa **30,000 objects**
  - **Enterprise Edition**: tối đa **500,000 objects**
- Phần setup khá chuyên sâu về **AD specific**, không cần đi sâu cho exam theo transcript.

## 2. Simple AD
- Là một **standalone managed directory**.
- Có **Active Directory compatible API**.
- **Không thể kết nối** với **on-premise Active Directory**.
- Điểm cần nhớ: đây là dạng directory đơn giản hơn, tách biệt khỏi hệ thống AD tại chỗ.

## 3. AD Connector
- Là một **proxy** để chuyển tiếp (**redirect**) các directory request tới **existing Microsoft Active Directory on-premise**.
- Được thiết kế theo 2 mức:
  - Connector cho tối đa **500 users**
  - Connector cho tối đa **5,000 users**
- Theo transcript, cần nhớ:
  - **AD Connector** là **proxy**
  - **first one supports MFA**
  - **the second one is just standalone**

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| AWS Managed Microsoft AD | AD tích hợp với AWS Cloud, có trust relationship với on-premise directory |
| Editions | Standard Edition: 30,000 objects; Enterprise Edition: 500,000 objects |
| Simple AD | Managed directory standalone, có Active Directory compatible API, không nối được với on-premise AD |
| AD Connector | Proxy chuyển request về Microsoft AD on-premise |
| AD Connector scale | 500 users hoặc 5,000 users |
| Cognito User Pool | Được redirect sang Cognito service, không tính vào Directory Services |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- **Managed Microsoft AD** = AD “chuẩn” nhất trong AWS, có thể **trust** với on-premise.
- **Simple AD** = **simple + standalone**, không connect được với on-premise AD.
- **AD Connector** = **proxy** chuyển request về on-premise AD.
- Nhớ kỹ từ khóa:
  - **trust relationship**
  - **standalone**
  - **proxy**
  - **on-premise Active Directory**
- **Amazon Cognito User Pool** không phải directory service trong ngữ cảnh bài học này.

## ✅ Kết luận
- AWS Directory Services trong bài này xoay quanh 3 dịch vụ chính: **AWS Managed Microsoft AD**, **Simple AD**, và **AD Connector**.
- Điểm mấu chốt để ôn thi là phân biệt:
  - dịch vụ nào **managed AD**
  - dịch vụ nào **standalone**
  - dịch vụ nào chỉ là **proxy** cho AD on-premise
