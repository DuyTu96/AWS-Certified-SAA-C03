# 269. Machine Learning Summary

## 🎯 Giới thiệu
- Phần này tổng hợp các **AWS Machine Learning services** cần nhớ cho kỳ thi.
- Trọng tâm là nhận diện đúng **service name** và **use case** tương ứng.

## 1. Các dịch vụ ML chính trên AWS
- **Rekognition**: dùng để **face detection**, **labeling**, và **celebrity recognition**.
- **Transcribe**: chuyển **audio** thành **text**, ví dụ tạo **subtitles**.
- **Polly**: ngược lại với Transcribe, dùng **text** để tạo **audio**.
- **Translate**: dùng cho **translation**.
- **Lex**: xây dựng **conversational bots / chatbots**.
  - Kết hợp với **Connect** có thể tạo **cloud contact center**.
- **Comprehend**: dùng cho **natural language processing**.
- **SageMaker**: dịch vụ **machine learning** đầy đủ tính năng, dành cho **developer** và **data scientist**.
- **Kendra**: **ML-powered document search engine**.
- **Personalize**: cung cấp **real-time personalized recommendations** cho khách hàng.
- **Textract**: phát hiện và trích xuất **text** và **data** từ nhiều loại **documents**.

## 2. Điểm cần nhớ để ôn thi
- AWS có một danh sách dịch vụ ML khá dài, nên cần nhớ theo **chức năng** thay vì học thuộc máy móc.
- Nhiều câu hỏi thi sẽ hỏi theo kiểu:
  - cần **speech-to-text** thì chọn **Transcribe**
  - cần **text-to-speech** thì chọn **Polly**
  - cần **chatbot** thì chọn **Lex**
  - cần **document search** thì chọn **Kendra**
  - cần **extract text from documents** thì chọn **Textract**
- Mục tiêu là nhận diện nhanh **keyword** trong đề để map đúng sang dịch vụ.

## 3. Từ khóa quan trọng
- **Rekognition**: face detection, labeling, celebrity recognition
- **Transcribe**: audio to text, subtitles
- **Polly**: text to audio
- **Translate**: translation
- **Lex**: chatbot, conversational bots
- **Connect**: cloud contact center
- **Comprehend**: natural language processing
- **SageMaker**: fully featured machine learning service
- **Kendra**: ML-powered document search engine
- **Personalize**: real-time personalized recommendations
- **Textract**: extract text and data from documents
