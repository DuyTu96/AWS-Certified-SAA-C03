# 238. ElastiCache

## 🎯 Giới thiệu
Amazon ElastiCache là dịch vụ managed dành cho **Redis** hoặc **Memcached** trên AWS. Có thể xem đây là một lựa chọn tương tự như **RDS**, nhưng áp dụng cho **cache technology**.

- Cache là một **in-memory data store**
- Mục tiêu chính: đạt **sub-millisecond latency** khi đọc dữ liệu
- Khi dùng ElastiCache, bạn cần **provision** một **ElastiCache instance type** cho cache của mình

## 1. 🔧 Tính năng chính
Nếu dùng **Redis** với ElastiCache, transcript nhắc đến các khả năng sau:

- **Clustering**
- **Multi AZ**
- **Read Replicas** để hỗ trợ **sharding**
- Bảo mật qua **IAM**
- Dùng **security groups**
- **KMS at-rest encryption**
- **Redis Authentication**

Ngoài ra, ElastiCache còn có các tính năng quản trị giống RDS:

- **Backups**
- **Snapshots**
- **Point-in-time restore**
- **Managed and scheduled maintenance**

## 2. 🚀 Điểm cần nhớ cho kỳ thi
Điểm quan trọng nhất:

- Khi dùng **Amazon ElastiCache** để cache dữ liệu trên ứng dụng hoặc kết hợp với **RDS**, bạn **phải thay đổi application code** để tận dụng ElastiCache
- Vì vậy, nếu đề thi hỏi về một giải pháp caching **không yêu cầu code change**, thì **ElastiCache không phải lựa chọn phù hợp**

## 3. 📌 Use cases
ElastiCache phù hợp cho các trường hợp:

- **Key/Value store**
- Cache cho các **database queries** khi có truy cập đọc thường xuyên
- Lưu **session data** cho người dùng website

Lưu ý thêm:

- Bạn **không thể dùng SQL** trên ElastiCache

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Ghi nhớ: **ElastiCache = cache in-memory, không phải database**
- Nếu đề bài nói **cần caching nhưng không muốn đổi code**, hãy loại **ElastiCache**
- Nếu thấy nhu cầu:
  - cache **database queries**
  - lưu **session data**
  - cần **Redis** với **Clustering / Multi AZ / Read Replicas**
  
  thì nghĩ đến **ElastiCache**
- Nhớ cụm từ: **“cache thì phải sửa code”** khi dùng ElastiCache

## ✅ Kết luận
**Amazon ElastiCache** là dịch vụ managed cache cho **Redis** và **Memcached**, giúp tăng tốc truy cập dữ liệu với độ trễ rất thấp. Đây là lựa chọn tốt cho **Key/Value cache**, **database query caching**, và **session storage**, nhưng cần lưu ý rằng ứng dụng phải được chỉnh sửa để tích hợp với ElastiCache.
