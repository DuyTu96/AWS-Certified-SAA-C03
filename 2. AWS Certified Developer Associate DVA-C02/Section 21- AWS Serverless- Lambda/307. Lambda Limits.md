# 307. Lambda Limits

## 🎯 Giới thiệu
- Bài này nói về các **Lambda Limits** quan trọng cần nhớ cho kỳ thi.
- Các giới hạn này là **per region**.
- Nội dung chia thành 2 nhóm chính:
  - **Execution limits**
  - **Deployment limits**

## 1. ⚙️ Execution Limits
- **Memory allocation**: từ **128 MB** đến **10 GB**
  - Tăng theo bước **64 MB**
  - Khi tăng memory thì Lambda cũng có thêm **vCPU**
- **Maximum execution time**: **900 seconds** = **15 minutes**
  - Nếu workload cần lâu hơn mức này thì không phù hợp với Lambda
- **Environment variables**: tối đa **4 KB**
- **/tmp storage**: tối đa **10 GB**
  - Dùng khi cần xử lý hoặc chứa các file lớn tạm thời
- **Concurrent executions**: tối đa **1000**
  - Có thể tăng thêm nếu gửi yêu cầu
  - Nên dùng **reserved concurrency** sớm

## 2. 🚚 Deployment Limits
- **Compressed .zip size**: tối đa **50 MB**
- **Uncompressed size**: tối đa **250 MB**
- Nếu file hoặc package lớn hơn giới hạn này thì cần cân nhắc cách khác
- Với file lớn, transcript gợi ý dùng **/tmp space**

## 3. 🧠 Ý nghĩa khi làm bài thi
- Nếu đề bài yêu cầu:
  - **30 GB RAM**
  - **30 minutes execution time**
  - **3 GB file**
- Thì đó là dấu hiệu cho thấy **Lambda không phải lựa chọn phù hợp**

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| Phạm vi áp dụng | **Per region** |
| Memory | **128 MB - 10 GB**, tăng theo **64 MB** |
| vCPU | Tăng khi tăng memory |
| Execution time | Tối đa **900 seconds / 15 minutes** |
| Environment variables | Tối đa **4 KB** |
| `/tmp` space | Tối đa **10 GB** |
| Concurrent executions | Tối đa **1000**, có thể request tăng |
| Compressed deployment package | Tối đa **50 MB** |
| Uncompressed deployment package | Tối đa **250 MB** |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Nhớ cụm số quan trọng: **128 MB - 10 GB**, **15 minutes**, **4 KB**, **10 GB**, **1000**, **50 MB**, **250 MB**
- Khi đề bài vượt các ngưỡng này, hãy nghĩ ngay đến việc **Lambda không phù hợp**
- Nếu cần xử lý file lớn tạm thời, nhớ đến **/tmp**
- Nếu lo ngại giới hạn concurrency, nhớ đến **reserved concurrency**

## ✅ Kết luận
- Lambda có các giới hạn rõ ràng về **memory, thời gian chạy, environment variables, storage, concurrency và deployment size**.
- Nắm chắc các con số này giúp trả lời nhanh các câu hỏi thi về việc **Lambda có phù hợp hay không**.
