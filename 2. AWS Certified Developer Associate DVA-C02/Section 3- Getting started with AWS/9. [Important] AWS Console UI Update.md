# 9. [Important] AWS Console UI Update

## 🎯 Giới thiệu

Bài học giải thích sự thay đổi giao diện của **AWS Console**. Nội dung chính là giúp người học không bị nhầm lẫn nếu giao diện trong video khác với giao diện đang thấy khi thực hành.

## 1. AWS Console có giao diện mới

AWS Console có thể hiển thị giao diện mới với:

- Nền trắng sáng hơn.
- Button màu xanh sáng.
- Button bo tròn hơn.
- Tổng thể hiện đại hơn.

Giao diện cũ có xu hướng:

- Màu xám nhiều hơn.
- Màu xanh nhạt hơn.
- Button vuông hơn thay vì bo tròn.

## 2. Khác biệt chủ yếu là về hình ảnh

Theo transcript, thay đổi này chủ yếu nằm ở mức **visual**.

📌 Điểm cần nhớ:

- Giao diện có thể nhìn khác so với video.
- Cách sử dụng về cơ bản vẫn giống nhau.
- Các thao tác trong bài học vẫn có thể áp dụng dù màu sắc hoặc hình dạng button thay đổi.

## 3. Khi nào cần chú ý?

Nếu AWS Console thay đổi nhiều hơn, ví dụ:

- Button bị chuyển vị trí.
- Luồng thao tác khác đáng kể.
- UI không còn giống về mặt usability.

Khi đó video có thể cần được cập nhật. Tuy nhiên, trong phạm vi transcript này, instructor nhấn mạnh rằng UI mới vẫn tương đương về cách sử dụng.

## 📊 Bảng tóm tắt

| Tiêu chí | Mô tả |
|----------|------|
| Nội dung bài học | Cập nhật về giao diện AWS Console |
| UI mới | Trắng sáng hơn, button xanh sáng, bo tròn |
| UI cũ | Xám hơn, xanh nhạt hơn, button vuông hơn |
| Mức độ ảnh hưởng | Chủ yếu ảnh hưởng visual |
| Usability | Về cơ bản vẫn giống nhau |

## ✅ Kết luận

Khi học và thực hành trên AWS Console, giao diện có thể khác với video do AWS cập nhật UI. Điều quan trọng là tập trung vào chức năng và luồng sử dụng, vì thay đổi trong transcript chủ yếu là về màu sắc và hình dạng button, không phải thay đổi lớn về usability.
