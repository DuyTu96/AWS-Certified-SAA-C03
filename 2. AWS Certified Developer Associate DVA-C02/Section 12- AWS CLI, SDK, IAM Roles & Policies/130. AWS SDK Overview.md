# 130. AWS SDK Overview

## 🎯 Giới thiệu
AWS SDK là bộ công cụ giúp ứng dụng của bạn thực hiện các hành động trực tiếp trên AWS bằng code, thay vì phải dùng CLI. Trong bài này, trọng tâm là nhận biết khi nào nên dùng SDK và mối liên hệ giữa SDK với CLI.

## 1. AWS SDK là gì? 🧩
- SDK = `Software Development Kit`.
- Dùng để gọi AWS services trực tiếp từ application code.
- Đây là cách làm việc với AWS ở mức code, không qua CLI.

## 2. AWS SDK có những ngôn ngữ nào? 🌐
- AWS có official SDK cho nhiều ngôn ngữ:
  - Java
  - .NET
  - Node.js
  - PHP
  - Python
  - Go
  - Ruby
  - C++
- Danh sách này có thể còn mở rộng theo thời gian.

## 3. SDK trong thực tế AWS 🔧
- SDK được dùng khi bạn gửi `API calls` đến các AWS services.
- Ví dụ được nhắc trong transcript:
  - `DynamoDB`
  - `Amazon S3`
- `AWS CLI` thực chất được viết bằng Python và dùng `Boto3 SDK`.
- Trong course, SDK sẽ được gặp nhiều hơn khi học `Lambda functions`.

## 📊 Bảng tóm tắt
| Tiêu chí | Mô tả |
|----------|------|
| Mục đích | Gọi AWS services trực tiếp từ application code |
| Cách dùng | Thực hiện `API calls` thay vì dùng CLI |
| Ngôn ngữ hỗ trợ | Java, .NET, Node.js, PHP, Python, Go, Ruby, C++ |
| Liên hệ với CLI | CLI dùng Python SDK `Boto3` |
| Ví dụ dịch vụ | DynamoDB, Amazon S3 |
| Khi gặp trong course | Đặc biệt khi học `Lambda functions` |

## 💡 Mẹo ghi nhớ cho kỳ thi AWS
- Nhớ rằng `SDK` là lựa chọn khi cần gọi AWS từ code của ứng dụng.
- `CLI` không phải công cụ duy nhất để làm việc với AWS; CLI cũng dựa trên SDK.
- Nếu đề bài nhắc đến ứng dụng gọi `DynamoDB` hoặc `S3` bằng code, hãy nghĩ đến `SDK`.
- Từ khóa cần nhớ: `SDK`, `API calls`, `Boto3`, `CLI`, `Lambda`.

## ✅ Kết luận
AWS SDK cho phép ứng dụng tương tác trực tiếp với AWS services bằng code. Đây là kiến thức nền tảng quan trọng cho AWS exam, đặc biệt khi làm việc với `Lambda` và các thao tác API trên dịch vụ như `DynamoDB` và `S3`.
